## Understand Linked Lists
Unlike an array which requests a single, massive block of contiguous memory, a Linked List requests tiny, isolated blocks of memory (called "Nodes") scattered wherever there is free space in the system.

* Singly Linked List: Each node contains the actual data (the Task) and a single memory pointer (reference) to the next node in the chain. You can only traverse this list forward.

* Doubly Linked List: Each node contains the data, a pointer to the next node, and a pointer to the previous node. This allows you to traverse the list both forwards and backwards, but it requires more memory to store the extra pointers.
