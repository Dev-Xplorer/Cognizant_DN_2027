## Project Setup
To set up this project locally, we will use a standard Java directory structure and initialize it for version control:

1. Create the Workspace: Create a root directory named InventoryManagementSystem to house the project.

2. Initialize Git: Initialize a local Git repository in the root directory to track changes before pushing to GitHub.

3. Structure the Codebase: Create a src (source) folder inside the root directory to keep the code organized.

4. Create the Classes: Inside the src folder, create the data model (Product.java) alongside the main execution class (InventoryManagement.java).
